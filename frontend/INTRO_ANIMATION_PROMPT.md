# Промпт: Новая Intro-анимация «Welcome to StrategyBRIX»

## Контекст

Приложение — **Onboarding Portal** (ранее Security Portal). При каждом открытии показывается короткая intro-анимация. Нужно заменить текущую на новую с текстом **«Welcome to StrategyBRIX»** и трансформацией. Логотип компании будет предоставлен отдельно.

---

## Технические параметры (обязательны)

### Стек и зависимости
- **React 18** + **Vite**
- **framer-motion** ^12.x — для анимаций
- **Tailwind CSS** — для стилей

### Компонент
- **Файл:** `frontend/src/components/Intro.jsx`
- **Props:** `{ onComplete }` — callback, вызывается по окончании анимации
- **Интеграция:** компонент рендерится в `App.jsx` при `showIntro === true`

### Обязательный контракт
```jsx
export default function Intro({ onComplete }) {
  useEffect(() => {
    const timer = setTimeout(() => {
      onComplete();
    }, 4500);  // ← Длительность: 4.5 секунды (не менять)
    return () => clearTimeout(timer);
  }, [onComplete]);
  // ...
}
```

### Длительность
- **Общая длительность:** 4500 мс (4.5 сек)
- Таймер `setTimeout(onComplete, 4500)` обязателен — иначе переход на Login/Dashboard сломается.

### Layout и позиционирование
- **Контейнер:** `fixed inset-0 z-50` — полноэкранный оверлей
- **Фон:** `bg-[#050B14]` (тёмный)
- **Выравнивание:** `flex items-center justify-center`
- ** overflow:** `overflow-hidden`

### Стиль приложения

- Шрифты: системные, sans-serif, antialiased
- Текст: `text-white`, `uppercase` для заголовков, `font-mono` для подписей

1. **Текст:** «Welcome to StrategyBRIX» (или «Welcome to StrategyBRIX.» — по желанию)

## Референс текущей анимации

Текущая структура (для понимания, не копировать):
- Фон: radial-gradient + linear-gradient
- Сетка на полу (perspective, motion)
- Частицы (20 шт, движение вверх)
- Центральный блок: иконка щита + текст «Security Portal» + «INITIALIZING»
- Этапы: draw (1.5s) → fill (1s) → scan (2s loop) → exit (4.5s)

## Задание для программиста

Подготовить **3 варианта** intro-анимации:

1. **Вариант A** — один визуальный стиль (например, минималистичный)
2. **Вариант B** — другой стиль (например, более «кибер»/tech)
3. **Вариант C** — третий стиль (например, корпоративный/элегантный)

В каждом варианте:
- Текст: «Welcome to StrategyBRIX»
- Placeholder под логотип (например, `div` с размерами или `img` с заглушкой)
- Трансформация/анимация появления
- Соблюдение всех технических параметров выше
- Длительность 4.5 сек, вызов `onComplete()` в конце
